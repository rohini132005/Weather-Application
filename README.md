# 🌦️ Weather Forecast Application

## 📌 Project Overview

The Weather Forecast Application is a web-based application that provides real-time weather information for any city. It fetches live weather data from an external API and displays details such as temperature, humidity, wind speed, and weather conditions. The application is designed with a simple and user-friendly interface for easy access to weather updates.

---

## 🎯 Objectives

* To provide real-time weather information
* To allow users to search weather by city name
* To display accurate and up-to-date weather data
* To enhance user experience with a clean UI

---

## ⚙️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **API:** OpenWeather API
* **Tools:** GitHub

---

## ✨ Features

* 🌍 Search weather by city name
* 🌡️ Display current temperature
* ☁️ Show weather conditions (Cloudy, Sunny, Rainy, etc.)
* 💧 Humidity and wind speed details
* 📱 Responsive design for all devices

---

## 🏗️ System Modules

1. **User Input Module**
   Allows users to enter city name

2. **API Integration Module**
   Fetches weather data from OpenWeather API

3. **Display Module**
   Shows weather details on the screen

---

## ▶️ How to Run the Project

1. Clone or download the repository
2. Extract the project folder
3. Get API key from OpenWeather
4. Add the API key in the JavaScript file
5. Open `index.html` in a browser

---

## 🔑 API Configuration

* Sign up at https://openweathermap.org
* Generate your API key
* Replace in code:

```javascript
const apiKey = "YOUR_API_KEY";
```
